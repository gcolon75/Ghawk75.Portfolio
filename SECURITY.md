# Security Policy

## Supported Versions

This is a portfolio website project. Security updates are applied to the main branch.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this portfolio website, please report it by:

1. **DO NOT** open a public issue
2. Email the repository owner directly (check GitHub profile for contact information)
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

You can expect:
- Initial response within 48 hours
- Regular updates on the status of your report
- Credit in the fix (if desired and appropriate)

## Security Scanning

This repository uses:
- **Dependabot**: Automated dependency updates and security alerts
- **npm audit**: Regular dependency vulnerability checks

## Dependencies

We strive to keep all dependencies up to date. If you notice an outdated or vulnerable dependency:
- Check if there's already a Dependabot PR open
- If not, please open an issue or PR to update it

## Best Practices

When contributing to this project:
- Never commit secrets, API keys, or credentials
- Keep dependencies updated
- Follow React security best practices
- Use HTTPS for all external resources
